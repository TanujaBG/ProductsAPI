# 🧑‍🏫 Session prompts — 7-day .NET/Azure track

Copy-paste-ready prompts, one per topic. Each is **self-contained** so a fresh chat session starts with full context.

---

## Day 1 — Topic 1: C# & ASP.NET Core fundamentals
> I'm preparing to become a .NET full-stack developer expert on Azure in 7 days. Today is Day 1. Teach me modern C# (records, pattern matching, async/await, LINQ, nullable reference types, dependency injection) and ASP.NET Core fundamentals (middleware pipeline, hosting model, configuration, DI container). Give me: a concise explanation, a hands-on coding exercise I build step by step, common pitfalls, and 10 interview questions with answers.

## Day 1 — Topic 2: Building REST APIs with ASP.NET Core
> Continuing my 7-day .NET/Azure prep (Day 1, second topic). Teach me to build production-grade REST APIs in ASP.NET Core: controllers vs minimal APIs, model validation, API versioning, error handling with ProblemDetails, filters, and OpenAPI/Swagger. Give me a hands-on API I build incrementally, best practices, and 10 interview questions with answers.

---

## Day 2 — Topic 3: EF Core + Azure SQL
> Day 2 of my 7-day .NET full-stack Azure prep. Teach me data access with EF Core against SQL Server/Azure SQL: DbContext, migrations, relationships, LINQ queries, tracking vs no-tracking, transactions, and performance tuning (N+1, indexes, compiled queries). Give me a hands-on data layer, pitfalls, and 10 interview questions with answers.

## Day 2 — Topic 4: AuthN/AuthZ with Microsoft Entra ID
> Day 2, second topic of my 7-day .NET/Azure prep. Teach me authentication and authorization: JWT, OAuth2, OpenID Connect, and integrating Microsoft Entra ID (Azure AD) with an ASP.NET Core API and client. Cover roles, policies, scopes, and securing endpoints. Give me a hands-on secured API, common mistakes, and 10 interview questions with answers.

---

## Day 3 — Topic 5: Frontend integration (React/Angular/Blazor)
> Day 3 of my 7-day .NET full-stack Azure prep. Teach me to build a frontend that consumes my .NET API. Let's use React (but note Blazor alternatives): project setup, calling APIs, auth token handling, state management, and CORS. Give me a hands-on UI wired to the API, best practices, and 10 interview questions with answers.

## Day 3 — Topic 6: Azure hosting options
> Day 3, second topic of my 7-day .NET/Azure prep. Compare Azure hosting options for .NET apps: App Service, Container Apps, AKS, and Azure Functions — when to use each, scaling, pricing, and deployment. Give me a hands-on deployment of my API to App Service and a containerized deploy, a decision matrix, and 10 interview questions with answers.

---

## Day 4 — Topic 7: Azure Storage (Blob, Queue, Table)
> Day 4 of my 7-day .NET full-stack Azure prep. Teach me Azure Storage: Blob, Queue, and Table storage — SDK usage from .NET, common patterns, SAS tokens, and lifecycle management. Give me hands-on code for uploading blobs and processing a queue, best practices, and 10 interview questions with answers.

## Day 4 — Topic 8: Event-driven design (Service Bus / Event Grid)
> Day 4, second topic of my 7-day .NET/Azure prep. Teach me event-driven architecture on Azure: Service Bus (queues, topics, subscriptions) vs Event Grid vs Event Hubs — when to use each. Cover messaging patterns, retries, dead-lettering, and idempotency. Give me a hands-on publisher/consumer in .NET, pitfalls, and 10 interview questions with answers.

---

## Day 5 — Topic 9: Caching & performance
> Day 5 of my 7-day .NET full-stack Azure prep. Teach me caching and performance: Azure Cache for Redis, in-memory and distributed caching, response caching, output caching, async best practices, and horizontal scaling. Give me hands-on caching in my API, measurement techniques, and 10 interview questions with answers.

## Day 5 — Topic 10: Observability
> Day 5, second topic of my 7-day .NET/Azure prep. Teach me observability: Application Insights, Log Analytics, structured logging with ILogger, distributed tracing (OpenTelemetry), metrics, and alerts. Give me hands-on instrumentation of my .NET API, dashboards/queries (KQL), and 10 interview questions with answers.

---

## Day 6 — Topic 11: CI/CD
> Day 6 of my 7-day .NET full-stack Azure prep. Teach me CI/CD for .NET on Azure using both GitHub Actions and Azure DevOps Pipelines: build, test, containerize, and deploy to Azure. Cover environments, secrets, approvals, and blue-green/slot deployments. Give me a hands-on pipeline YAML I build step by step, best practices, and 10 interview questions with answers.

---

## Day 7 — Topic 12: IaC & architecture
> Day 7 of my 7-day .NET full-stack Azure prep. Teach me Infrastructure as Code with Bicep (and Terraform basics) plus cloud architecture: provisioning App Service/SQL/Storage, secure design (Key Vault, managed identities, networking), the Well-Architected Framework, and cost optimization. Give me a hands-on Bicep deployment, a reference architecture for a full-stack app, and 10 interview questions with answers.
